# 🛍️ Productify

## 🚀 Introduction
Productify is a full-stack PERN (PostgreSQL, Express, React, Node.js) marketplace where users can buy and sell tech products. It includes a complete authentication system, product management, cart & wishlist features, and a modern themed UI.

---

## ⚙️ Technologies Used
- React.js
- Node.js
- Express.js
- PostgreSQL
- DaisyUI
- REST API
- JWT Authentication

---

## ✨ Features
- Full authentication system (Signup, Login, Email OTP)
- Forgot & Reset Password functionality
- Add to Cart & Wishlist
- Product search & filter system
- Product detail page with comments
- Users can add & delete their own comments
- Sell products (Add, Edit, Delete)
- Theme switching using DaisyUI
- Responsive UI

---

## 🧠 Project Flow
- Homepage with hero section
- Product listing with search & filters
- Users can view product details
- Add to cart, wishlist, and place fake orders
- Comment system for each product
- Authentication required for selling products
- Users manage their own products and comments
